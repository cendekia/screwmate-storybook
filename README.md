# Screwmate Storybook

It is a storybook for screwmate project application. Screwmate stack will be using NextJS and TailwindCSS. So the storybook should be written using ReactJS component.

## Stacks:
* NextJS
* TailwindCSS
* ExpressJS
* SocketIO
